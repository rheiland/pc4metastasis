GUI Data Viewer

  • Macroscale model tab:   visualize the model selection of macroscale models.
    • Model Selection Breast and Model Selection Metastasis headers:   click on to reveal or hide the contents of model selection.
      • Observational Data tab:   display observational model data: Red points/bar = non-hypoxic cell average/standard deviation. Green points/bar = post-hypoxic cell average/standard deviation.
      • Model Candidates tab:   show mathematical structures of model candidates and conditional assumptions.
      • Parameters tab:   show parameter ranges of mathematical models.
      • Calibration tab:   show the model calibration for each model. You can slide the model ID to look at the parameters table and hypotheses for each model. You can choose the plot option in the dropdown options, to see details of calibration.
        • Solution option:   display of cell population from macroscale model (lines) and observational data (points/bar).
        • MCMC chains option:   display of the multiple chains (30 chains, 1000 samples each, and burn-in 100 samples) of Markov Chain Monte Carlo method.
        • Posterior dist option:   posterior marginalized distribution plot for each parameter.
      • Model Selection tab:   display the model selection using the weight of information criteria (Akaike, second-order Akaike, Bayesian), Complexity vs Accuracy plot, table with information for each model, and table with relative importance of each hypothesis.
  • Macro to Micro tab:   display the mathematical structure and hypotheses of models from the macroscale dynamics. You can visualize this data for with the scaling factor, and intravasation/extravasation rates used to propose the microscale method.
  • Microscale model tab:   display the results of the microscale model for two selected models (model 68 and 89).
    • Model 68 and Model 89 tabs:   show results of two hypotheses. H1: different proliferation rates and equal death rates, and H2: different proliferation rates and equal death rates for non-hypoxic and post-hypoxic cells. You can change view options.
      • Population tab:   display cell population of the non-hypoxic and post-hypoxic cells on macroscale and microscale (average and replicates of 10 replicates) models.
      • 3D images tab:   show snapshots of 3D images over 40 simulation days. You can select the day.
      • Clusters tab:   bar graph of clusters with more cells than a specified value (you can edit it). Each bar represents one cluster and shows the fraction of non-hypoxic and post-hypoxic cells.
      • Ki67 stain tab:   display the five-number summary of the Ki67 stain from each time (you can edit).

This model and cloud-hosted demo are part of a collaboration between Dr. Macklin's lab and Dr. Gilkes's lab, through of a project granted by Jayne Koskinas Ted Giovanis (JKTG) Foundation for Health and Policy and the Breast Cancer Research Foundation (BCRF). The model is built using PhysiCell: a C++ framework for multicellular systems biology.
